# IoT Peltier Temperature Control

Tímový projekt: regulácia teploty kvapaliny pomocou Peltierovho článku.

## Časti projektu

- Arduino/ESP32 riadenie
- Flask backend
- Frontend grafy a ovládanie
- SQL databáza a JSON export
- Identifikácia systému
- PID regulácia
- ThingsBoard integrácia

## Módy riadenia

- Mode 0: bez regulácie
- Mode 1: PID cez TEC
- Mode 2: PID cez pumpu
